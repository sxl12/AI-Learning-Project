# AI Learning Project

## 项目目标
本项目旨在学习AI大模型的基础知识、代理（Agent）应用开发以及模型微调。通过理论与实践结合，深入理解这些技术，并进行实际开发。

## 学习模块
1. **大模型基础**：学习大模型的原理和应用。
2. **Agent 应用开发**：开发智能代理完成各种任务。
3. **模型微调**：将预训练的大模型微调至特定任务。
